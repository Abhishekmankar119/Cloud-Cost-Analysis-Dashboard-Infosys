Project Overview:

This project focuses on analyzing cloud infrastructure costs across multiple cloud providers (AWS, Azure, and GCP). The dashboard provides insights into cloud spending patterns, billing models, regional cost distribution, resource utilization efficiency, and month-over-month cost trends. The objective is to help stakeholders monitor cloud expenses and identify potential cost optimization opportunities.

Business Problem:

Organizations often face challenges in tracking cloud expenditure across different providers, regions, and billing models. Without centralized reporting, it becomes difficult to identify cost drivers, monitor spending trends, and make informed decisions regarding cloud resource allocation.

Developed an interactive Power BI dashboard to:

Monitor overall cloud spending across AWS, Azure, and GCP.
Analyze costs by region, service, and billing mode (On-Demand, Reserved, Spot).
Track month-over-month cloud cost growth and spending trends.
Measure cost efficiency and identify optimization opportunities.
Provide a centralized view of cloud cost data for decision-making.

Key Features:
Executive KPI Summary
Multi-Cloud Cost Analysis
Regional Cost Distribution Map
Billing Mode Comparison
Cost Efficiency Metrics
Dynamic Filters and Drill-Down Analysis
Monthly Trend Analysis
